## 这份SOP是干什么的

把你的产品发售过程，变成一场"朋友圈连续剧"。用户从围观到付费，全程被势能推着走。

核心原理：**连环圈+养成感+势能堆叠**

## 8种圈型速查

| 圈型 | 一句话功能 | 使用频率 |
|------|------------|----------|
| 连环圈 | 串联事件因果，官宣起势 | 鸣笛期/揭秘期高频 |
| 晒单圈 | 展示成交力，但要晒故事不晒数字 | 发售期/累势期 |
| 价值圈 | 输出认知，制造深度连接 | 全阶段穿插 |
| 产品圈 | 产品信息、价格、倒计时 | 揭秘期/发售期高频 |
| 证言圈 | 学员反馈，口碑展示 | 发售期/累势期高频 |
| 价值观圈 | 人格魅力，筛选对的人 | 预热期/累势期 |
| 团队圈 | 展示交付能力，让用户放心 | 发售期/累势期 |
| 预告圈 | 勾起好奇，留悬念不说透 | 鸣笛期/预热期 |

## 全阶段节奏总览

```
筹备期（2-4周） → 鸣笛期（1-2周） → 预热期（3-7天） → 揭秘期（1-3天） → 发售期（3-7天） → 累势期（2-4周）
   ↑___________________________循环：下一次发售从累势期尾部开始___________________________↓
```

## 贯穿全流程的8条心法

1. **克制即高级**——最高级的营销是克制，少即是多，不嚣而喧，不销而销
2. **真诚破一切**——真诚是解决所有异议的唯一方法，别用套路找借口
3. **能量 > 文笔**——靠能量发圈，比靠文笔、晒单更有成交力
4. **养成感 > 广告感**——用户见证你的过程，比看到你的结果更有参与感
5. **损失厌恶是核武器**——限额+倒计时+退费政策，三重刺激推动决策
6. **连环即因果**——每条圈之间要有逻辑关系，有因有果
7. **日常养IP > 发售期突击**——发售期的势能，来自日常积累
8. **晒故事不晒数字**——晒"感觉是你亏了"，比晒收款截图高级十倍

## 每次发圈前过一遍的自检清单

- [ ] 和前一条圈有逻辑关联吗？（连环感）
- [ ] 能独立成立吗？（不依赖上下文也能看懂）
- [ ] 核心信息够精简吗？（一条圈一个点）
- [ ] 是在"卖产品"还是"讲故事/传价值"？（后者优先）
- [ ] 能引发用户思考或好奇吗？
- [ ] 克制感如何？（是否太用力）
- [ ] 这条圈属于哪种圈型？定位清晰吗？
